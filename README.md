## 開発

| コマンド               | 説明                             |
| :--------------------- | :------------------------------- |
| `npm run dev`          | 開発用サーバを起動する           |
| `npm run build`        | ビルドする                       |
| `npm run start`        | ビルドした成果物を起動する       |
| `npm run lint`         | リントを実行する                 |
| `npm run format:check` | フォーマットのチェックを実行する |
| `npm run test`         | テストを実行する                 |
